# Module 3: Daily Life & Activities

## Title: Talking About Your Life in ASL

### Lecture Content

Now let's learn signs for everyday life - family, food, activities, and daily routines. These signs form the foundation for most conversations.

#### 1. Family Members

**Visual Guide - Family Signs:**

```
FAMILY:
  Option 1: F Shapes Circle      Option 2: Touch & Separate
    ┌─────┐  ┌─────┐              ┌─────┐  ┌─────┐
    │ ║║║ │  │ ║║║ │              │ │││ │  │ │││ │
    │ ║║║ │○│ ║║║ │              │ │││ │→←│ │││ │
    └─────┘  └─────┘              └─────┘  └─────┘
  (circle around)                (touch, separate)

MOTHER/MOM:          FATHER/DAD:
    ┌─────┐              ┌─────┐
    │ │││ │              │ │││ │
    │ │││ │ tap tap      │ │││ │ tap tap
    └─╯─┘              └─╯─┘
  (thumb on chin)     (thumb on forehead)

SISTER:              BROTHER:
    ┌─────┐              ┌─────┐
    │ │   │              │ │   │
    │ │   │ tap          │ │   │ tap
    └─╯─┘              └─╯─┘
  (tap chin, then      (tap forehead,
   tap other hand)      then tap other)

DAUGHTER:            SON:
  MOM ↓              DAD ↓
  (smaller)           (smaller)

GRANDMOTHER:         GRANDFATHER:
  MOM → →            DAD → →
  (forward twice)    (forward twice)

AUNT:                UNCLE:                COUSIN:
    ┌─────┐              ┌─────┐              ┌─────┐
    │ ║║║ │              │ ││  │              │ ╭─╮ │
    │ ║║║ │ ○           │ ││  │ ○            │ │ │ │ tap
    └─╯─┘              └─╯─┘              └─╯─┘
  (A shape, circle    (U shape, circle    (C shape,
   near cheek)         near forehead)      tap head)
```

**FAMILY:**
- Both hands, F handshapes, circle around each other
- Or: Both hands, open, fingertips touch, then separate

**MOTHER/MOM:**
- Open hand, thumb on chin, tap twice
- Or: M handshape on chin

**FATHER/DAD:**
- Open hand, thumb on forehead, tap twice
- Or: F handshape on forehead

**SISTER:**
- Index finger, tap chin (like MOM), then tap dominant hand's index finger
- Or: S handshape, tap chin, then tap other hand

**BROTHER:**
- Index finger, tap forehead (like DAD), then tap dominant hand's index finger
- Or: B handshape, tap forehead, then tap other hand

**DAUGHTER:**
- MOM sign, then move hand down (smaller)

**SON:**
- DAD sign, then move hand down (smaller)

**GRANDMOTHER:**
- MOM sign, then move hand forward twice (away from body)

**GRANDFATHER:**
- DAD sign, then move hand forward twice

**AUNT:**
- A handshape, circle near cheek

**UNCLE:**
- U handshape, circle near forehead

**COUSIN:**
- C handshape, tap side of head

#### 2. Food and Drinks

**Visual Guide - Food Signs:**

```
EAT:                DRINK:               WATER:
    ┌─────┐              ┌─────┐              ┌─────┐
    │ │││ │              │ ╭─╮ │              │ │││ │
    │ │││ │ →           │ │ │ │ →           │ │││ │ tap
    └─────┘              └─╯ └─┘              └─╯─┘
  (to mouth)           (C shape to mouth)  (W shape, chin)

FOOD:                HUNGRY:              THIRSTY:
    ┌─────┐              ┌─────┐              ┌─────┐
    │ │││ │              │ ╭─╮ │              │ │   │
    │ │││ │ tap tap      │ │ │ │ ↓            │ │   │ ↓
    └─╯─┘              └─╯─┘              └─────┘
  (tap mouth twice)   (C shape down chest) (finger down throat)

COFFEE:              MILK:                BREAD:
    ┌─────┐              ┌─────┐              ┌─────┐  ┌─────┐
    │ ║║║ │              │ │││ │              │ │││ │  │ │││ │
    │ ║║║ │ ○           │ │││ │ squeeze      │ │││ │→│ │││ │
    └─╯─┘              └─────┘              └─────┘  └─────┘
  (S shape, rotate)    (squeeze motion)     (slice across palm)

PIZZA:               APPLE:                BANANA:
    ┌─────┐              ┌─────┐              ┌─────┐
    │ ║║║ │              │ ║║║ │              │ │   │
    │ ║║║ │ ○           │ ║║║ │ twist        │ │   │ ↓
    └─────┘              └─╯─┘              └─────┘
  (P shape, circle)    (knuckle on cheek)  (peel down motion)
```

**EAT:**
- Flat hand, fingertips to mouth, bring to mouth (like eating)

**DRINK:**
- C handshape, bring to mouth (like drinking from cup)

**WATER:**
- W handshape, tap chin

**FOOD:**
- Flat hand, fingertips together, tap mouth twice

**BREAKFAST:**
- EAT + MORNING

**LUNCH:**
- EAT + NOON (flat hand, palm down, moves across)

**DINNER:**
- EAT + EVENING

**HUNGRY:**
- C handshape, moves down chest (like stomach)

**THIRSTY:**
- Index finger, moves down throat

**COFFEE:**
- S handshape, rotate near mouth (like grinding)

**MILK:**
- Squeeze motion (like milking a cow)

**BREAD:**
- Dominant hand slices across non-dominant palm

**PIZZA:**
- P handshape, moves in circle (like spreading sauce)

**APPLE:**
- Closed fist, knuckle on cheek, twist

**BANANA:**
- Index finger, peels down (like peeling banana)

#### 3. Hobbies and Activities
**LIKE:**
- Flat hand, palm on chest, moves forward

**LOVE:**
- Both hands, crossed on chest (hug yourself)

**ENJOY:**
- Both hands, rub together (like enjoying something)

**READ:**
- Both hands, palms facing each other, move apart (like opening book)

**WRITE:**
- Index and middle fingers, write in air

**DRAW:**
- Index finger, draws in air

**SING:**
- Flat hand, moves back and forth near mouth

**DANCE:**
- Both hands, move in dancing motion

**SWIM:**
- Both hands, alternate swimming motion

**RUN:**
- Both hands, L handshapes, alternate forward motion

**WALK:**
- Both hands, index and middle fingers, alternate walking motion

**PLAY:**
- Both Y handshapes, move back and forth

**SPORTS:**
- Both hands, S handshapes, cross and uncross

**MUSIC:**
- Both hands, move in rhythm (like conducting)

**MOVIE:**
- Both hands, form rectangle (like screen)

**GAME:**
- Both hands, A handshapes, move back and forth

#### 4. Daily Routines
**WAKE UP:**
- Both hands, open, move up (like opening eyes)

**SLEEP:**
- Flat hand, palm down, moves down in front of face

**SHOWER:**
- Both hands, rub body (like showering)

**BRUSH TEETH:**
- Index finger, brush motion near mouth

**GET DRESSED:**
- Both hands, move down body (like putting on clothes)

**GO:**
- Index finger, points forward, moves forward

**COME:**
- Index finger, points toward you, moves toward you

**HOME:**
- Flat hand, fingertips touch mouth, then touch cheek

**SCHOOL:**
- Both hands, clap together twice (like opening/closing books)

**WORK:**
- Both hands, S handshapes, tap together

**STORE:**
- Both hands, A handshapes, move forward (like pushing cart)

#### 5. Common Verbs
**SEE:**
- V handshape, fingertips near eyes, move forward

**LOOK:**
- V handshape, move toward object

**HEAR:**
- Index finger, points to ear

**UNDERSTAND:**
- Index finger, touches forehead, then moves forward

**KNOW:**
- Fingers touch forehead, then move forward

**THINK:**
- Index finger, circles near forehead

**REMEMBER:**
- Hand touches forehead, then moves to non-dominant hand (like storing)

**FORGET:**
- Hand touches forehead, then moves away (like wiping away)

**WANT:**
- Both hands, claw handshapes, pull toward body

**NEED:**
- X handshape, moves down (like pulling down)

**HAVE:**
- Both hands, palms up, move toward body

**GIVE:**
- Both hands, move from body outward

**TAKE:**
- Both hands, move toward body

#### 6. ASL Sentence Structure
ASL uses different word order than English:
- **Time** - **Topic** - **Comment**
- **Object** - **Subject** - **Verb** (often)

**Examples:**
- English: "I eat breakfast every morning"
- ASL: "MORNING, BREAKFAST I EAT" or "EVERY MORNING BREAKFAST I EAT"

- English: "I like pizza"
- ASL: "PIZZA I LIKE" or "I LIKE PIZZA"

**Note:** ASL is flexible. Context and facial expressions help clarify meaning.

#### 7. Describing Routines
**EVERY DAY:**
- Flat hand, palm down, moves forward repeatedly

**EVERY MORNING:**
- MORNING + EVERY

**USUALLY:**
- Y handshape, moves side to side

**SOMETIMES:**
- Index finger, moves up and down

**ALWAYS:**
- Index finger, circles continuously

**NEVER:**
- Open hand, moves across body (like cutting)

**Example sentence:**
"EVERY MORNING, I WAKE UP, SHOWER, EAT BREAKFAST, GO SCHOOL"

### Cultural Note
In Deaf culture, family often includes:
- Biological family
- Deaf school classmates (often called "Deaf family")
- Close friends in the community
- Mentors and role models

Family connections in the Deaf community can be very strong due to shared language and culture.
