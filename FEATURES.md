# School 2 - Enhanced Features

## Features Integrated from SalonApp

### 1. Theme System
- **Multiple Visual Themes**: Liquid Glass, Instagram-style, Minimal, Luxury, etc.
- **Theme Switcher**: Easy theme selection with preview
- **Dark/Light Mode**: System preference detection

### 2. UI Layout Options
- **Beauty Card Layout**: Instagram-style card layout with large images
- **Feed Layout**: Social media-style scrolling feed
- **Sidebar Layout**: Traditional sidebar navigation
- **Modern Layout**: Minimalist design with bottom navigation

### 3. AI Search Assistant
- **Natural Language Search**: Ask questions in plain English
- **Smart Search**: Searches courses, modules, quizzes, and content
- **Advanced Queries**: "Show me modules about logic", "Find quizzes I haven't completed"

### 4. Learning Potential Prediction
- **Potential Calculator**: Predicts your learning potential based on:
  - Current progress
  - Quiz scores
  - Study time
  - Module completion rate
- **Visual Progress**: Shows percentage of full potential reached
- **Motivational Insights**: "You're at 65% of your full potential!"
