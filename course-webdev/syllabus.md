### Course Description
This comprehensive 8-week course provides hands-on training in modern web development. Students will learn HTML, CSS, JavaScript, and essential web development tools to build responsive, interactive websites. The course emphasizes practical projects, real-world applications, and best practices for deploying websites. By the end, students will have a portfolio of working web projects.

### Learning Outcomes
Upon successful completion, students will be able to:
1. Write semantic HTML5 markup
2. Style websites with CSS3 and create responsive layouts
3. Add interactivity with JavaScript (ES6+)
4. Use Git for version control
5. Deploy websites to production
6. Debug and troubleshoot web applications
7. Understand web performance basics
8. Build a complete portfolio website

### Weekly Schedule

#### Week 1: HTML Foundations
- HTML5 semantic elements
- Forms and input types
- Accessibility basics
- Document structure
- First webpage project

#### Week 2: CSS Styling & Layout
- CSS selectors and specificity
- Box model and positioning
- Flexbox fundamentals
- CSS Grid basics
- Responsive design principles

#### Week 3: Advanced CSS & Responsive Design
- Media queries
- CSS variables and custom properties
- Animations and transitions
- Mobile-first approach
- Responsive portfolio page

#### Week 4: JavaScript Basics
- Variables, data types, operators
- Functions and scope
- Control flow (if/else, loops)
- DOM manipulation
- Interactive webpage project

#### Week 5: JavaScript Intermediate
- Arrays and objects
- ES6+ features (arrow functions, destructuring)
- Event handling
- Form validation
- Dynamic content project

#### Week 6: JavaScript Advanced & APIs
- Async JavaScript (Promises, async/await)
- Fetch API and HTTP requests
- Error handling
- Working with JSON data
- API integration project

#### Week 7: Version Control & Tools
- Git basics (init, commit, push)
- GitHub workflow
- Code organization
- Development tools (VS Code, browser DevTools)
- Project management

#### Week 8: Deployment & Portfolio
- Web hosting options
- Deployment process (Netlify/Vercel)
- Domain setup basics
- Performance optimization
- Final portfolio website deployment
