# Module 2: Greetings & Introductions

## Title: Meeting People in ASL

### Lecture Content

Now that you know the alphabet and basics, let's learn to greet people and introduce yourself! This module covers essential social signs you'll use in every conversation.

#### 1. Basic Greetings

**Visual Guide - Greeting Signs:**

```
HELLO/HI:
  Start:          Wave Motion:
    ┌─────┐         ┌─────┐    ┌─────┐    ┌─────┐
    │ │││ │   →     │ │││ │ →  │ │││ │ →  │ │││ │
    │ │││ │         │ │││ │    │ │││ │    │ │││ │
    └─────┘         └─────┘    └─────┘    └─────┘
  (palm forward)    (small friendly wave side to side)

GOOD MORNING:
  GOOD:                    MORNING:
    ┌─────┐                  ┌─────┐
    │ │││ │                  │ │││ │
    │ │││ │ → (away)         │ │││ │ → (across)
    └─────┘                  └─────┘
  (palm down, from mouth)   (palm up, across body)

GOODBYE:
  Option 1: Wave            Option 2: Move Away
    ┌─────┐    ┌─────┐         ┌─────┐
    │ │││ │ ↔ │ │││ │         │ │││ │ →
    │ │││ │    │ │││ │         │ │││ │
    └─────┘    └─────┘         └─────┘
  (side to side)            (away from body)
```

**HELLO/HI:**
- Wave your hand (open hand, palm facing forward)
- Small, friendly wave
- Can be done with a smile

**GOOD MORNING:**
- Sign GOOD (flat hand, palm down, moves away from mouth)
- Then MORNING (flat hand, palm up, moves across body)
- Or combine: GOOD + MORNING

**GOOD AFTERNOON:**
- GOOD + AFTERNOON (flat hand moves down from forehead)

**GOOD EVENING/NIGHT:**
- GOOD + NIGHT (flat hand, palm down, moves down)

**GOODBYE:**
- Open hand, palm facing forward
- Wave side to side
- Or: Open hand, fingers together, moves away from body

#### 2. Polite Expressions

**Visual Guide - Polite Signs:**

```
PLEASE:
  Start:          Circular Motion:
    ┌─────┐         ┌─────┐
    │ │││ │         │ │││ │
    │ │││ │    →    │ │││ │  (clockwise circle)
    └─────┘         └─────┘
  (palm on chest)  (circular motion)

THANK YOU:
  Start:          Movement:
    ┌─────┐         ┌─────┐
    │ │││ │         │ │││ │
    │ │││ │    →    │ │││ │ ↓
    └─────┘         └─────┘
  (fingertips      (forward and down)
   touch chin)

YOU'RE WELCOME:
  Start:          Movement:
    ┌─────┐         ┌─────┐
    │ │││ │         │ │││ │ →
    │ │││ │    →    │ │││ │
    └─────┘         └─────┘
  (palm up,        (toward person)
   on chest)

SORRY:
  Start:          Circular Motion:
    ┌─────┐         ┌─────┐
    │ ║║║ │         │ ║║║ │
    │ ║║║ │    →    │ ║║║ │  (circular rub)
    └─────┘         └─────┘
  (fist on chest)  (with sincere face)

EXCUSE ME:
  Start:          Movement:
    ┌─────┐         ┌─────┐
    │ │││ │         │ │││ │ →
    │ │││ │    →    │ │││ │
    └─────┘         └─────┘
  (fingertips      (forward)
   touch forehead)
```

**PLEASE:**
- Flat hand, palm on chest
- Move in circular motion (clockwise)
- Can be done once or repeated

**THANK YOU:**
- Flat hand, fingertips touch chin
- Move hand forward and down
- Or: Hand moves from chin outward

**YOU'RE WELCOME:**
- Flat hand, palm up, moves toward person
- Or: Hand moves from chest outward

**SORRY:**
- Closed fist, rub in circular motion on chest
- Facial expression shows sincerity

**EXCUSE ME:**
- Flat hand, fingertips touch forehead
- Move hand forward
- Used to get attention or apologize

#### 3. Introducing Yourself

**Visual Guide - Introduction Signs:**

```
MY NAME:
  MY:              NAME:
    ┌─────┐         ┌─────┐
    │ │││ │         │ ││  │
    │ │││ │         │ ││  │  tap tap
    └─────┘         └─────┘
  (palm on chest)  (H shape, tap twice)

I / ME / YOU:
  I:               YOU:              ME:
    ┌─────┐         ┌─────┐           ┌─────┐
    │ │   │         │ │   │           │ ↑   │
    │ │   │ →       │ │   │ →        │ ║║║ │ →
    └─────┘         └─────┘           └─────┘
  (point to        (point to         (thumb to
   your chest)      other person)     your chest)

NICE TO MEET YOU:
  NICE:            MEET:             YOU:
    ┌─────┐         ┌─────┐  ┌─────┐  ┌─────┐
    │ │││ │         │ │││ │  │ │││ │  │ │   │
    │ │││ │ ↓       │ │││ │→←│ │││ │  │ │   │ →
    └─────┘         └─────┘  └─────┘  └─────┘
  (brush down)     (hands come    (point to
                    together)      person)
```

**MY NAME [fingerspell name]:**
- MY: Flat hand, palm on chest
- NAME: H handshape (index and middle fingers), tap twice
- Then fingerspell your name

**I:**
- Point index finger to your chest

**YOU:**
- Point index finger to the person you're addressing

**ME:**
- Point thumb to your chest

**NICE TO MEET YOU:**
- NICE: Flat hand, palm down, brushes down chest
- MEET: Both hands, palms facing each other, come together
- YOU: Point to person

#### 4. Asking About Others

**Visual Guide - Question Signs:**

```
WHAT YOUR NAME?
  WHAT:            YOUR:            NAME:
    ┌─────┐         ┌─────┐         ┌─────┐
    │ │││ │         │ │   │         │ ││  │
    │ │││ │ ↔      │ │   │ →       │ ││  │ tap tap
    └─────┘         └─────┘         └─────┘
  (palm up,        (point to        (H shape)
   shake)           person)
  ╲╱ Eyebrows DOWN (WH-question)

HOW YOU?
  HOW:                         YOU:
    ┌─────┐  ┌─────┐           ┌─────┐
    │ │││ │  │ │││ │           │ │   │
    │ │││ │ ↑│ │││ │ ↓         │ │   │ →
    └─────┘  └─────┘           └─────┘
  (alternate up/down)         (point to person)
  ╲╱ Eyebrows DOWN

WHERE YOU FROM?
  WHERE:            YOU:            FROM:
    ┌─────┐         ┌─────┐         ┌─────┐
    │ │   │         │ │   │         │ │   │
    │ │   │ ↔      │ │   │ →       │ │   │ →
    └─────┘         └─────┘         └─────┘
  (shake side)      (point)         (move from
                                     location A→B)
```

**WHAT YOUR NAME?**
- WHAT: Open hand, palm up, shakes side to side
- YOUR: Point to person
- NAME: H handshape, tap twice
- **Facial expression:** Eyebrows down (WH-question)

**HOW YOU?**
- HOW: Both hands, palms up, move up and down alternately
- YOU: Point to person
- **Facial expression:** Eyebrows down

**WHERE YOU FROM?**
- WHERE: Index finger, shake side to side
- YOU: Point to person
- FROM: Index finger, moves from one location to another

#### 5. Responses
**I FINE:**
- I: Point to chest
- FINE: Flat hand, thumb up, moves up slightly

**I GOOD:**
- I: Point to chest
- GOOD: Flat hand, palm down, moves away from mouth

**I OKAY:**
- I: Point to chest
- OKAY: F-O handshape (thumb and index form circle), moves up

**I FROM [location]:**
- I: Point to chest
- FROM: Index finger moves from location
- Then fingerspell or sign location name

#### 6. Getting Attention
In Deaf culture, you get attention by:

```
Attention-Getting Methods:

1. Tapping Shoulder:        2. Waving:
      [You]                      ┌─────┐
        │                        │ │││ │
        ↓                        │ │││ │ ↔
      [Person]                   └─────┘
   (light tap, not hard)      (in line of sight)

3. Stomping Foot:            4. Flicking Lights:
      [Foot]                     [Light Switch]
        ↓                            ↕
      [Floor]                    (flick on/off)
   (creates vibrations)        (in group settings)

5. Eye Contact:
      [You] ←→ [Person]
   (maintain once established)
```

- **Tapping shoulder:** Light tap, not hard
- **Waving:** In person's line of sight
- **Stomping foot:** On floor (vibrations)
- **Flicking lights:** In group settings
- **Eye contact:** Once you have attention, maintain it

**Important:** Never throw objects or be aggressive. Always be respectful.

#### 7. Eye Contact and Turn-Taking
- **Maintain eye contact** during conversation
- **Don't watch hands** - watch the signer's face
- **Wait for turn** - don't interrupt
- **Use visual signals** to indicate you want to speak
- **Look away** when you're done signing

#### 8. Basic Personal Information
**AGE:**
- Sign the number, then move hand up (older) or down (younger)
- Or: Number + OLD

**STUDENT:**
- Both hands, palms down, move forward (like opening books)

**TEACHER:**
- Both hands, fingertips touch, then separate (like spreading knowledge)

**WORK:**
- Both hands, S handshape, tap together

**LIVE:**
- Both L handshapes, move down (like setting down roots)

#### 9. Conversation Flow
**Basic structure:**
1. Get attention
2. Greet
3. Introduce yourself
4. Ask about other person
5. Respond appropriately
6. Close conversation politely

**Example conversation:**
- A: HELLO
- B: HELLO
- A: MY NAME [fingerspell]
- B: NICE MEET YOU. MY NAME [fingerspell]
- A: HOW YOU?
- B: I FINE, THANK YOU. YOU?
- A: I GOOD, THANK YOU

### Exercises (Complete all 5)
1. **Greetings Practice:** Practice HELLO, GOODBYE, GOOD MORNING, GOOD NIGHT with appropriate facial expressions
2. **Polite Expressions:** Sign PLEASE, THANK YOU, YOU'RE WELCOME, SORRY, EXCUSE ME 10 times each
3. **Introduction:** Practice introducing yourself: "HELLO, MY NAME [your name], NICE MEET YOU"
4. **Questions:** Practice asking "WHAT YOUR NAME?" and "HOW YOU?" with proper facial expressions
5. **Conversation:** Write out and practice a 30-second introduction conversation

### Cultural Note
In Deaf culture, introductions often include:
- Your name (fingerspelled or name sign)
- Where you're from
- How you learned ASL
- Connection to Deaf community

Name signs are given by Deaf community members, not chosen by yourself. Don't create your own name sign!
