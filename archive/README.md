# Archive

This folder holds files moved out of the project root during the 2026-05 cleanup pass — not deleted, just removed from active view.

- `docs/` — 90+ legacy status/checklist/troubleshooting markdown files (FIX_RESEND_KEY_*, SAVE_RESEND_KEY_*, TASK_*_COMPLETE.md, etc). Most were intermediate progress notes that are now stale.
- `dev-html/` — one-off browser helper pages used during local debugging (save/extract Resend keys, force-reload Supabase). Not part of the deployed site.

If you find yourself needing one of these regularly, pull it back to the root and tell me — it shouldn't have been archived.
